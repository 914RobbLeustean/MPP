Return Requirements

To be eligible for a return, your item must meet these conditions:

Original Tags

All tags must be attached and intact

Unworn Condition

Items must be unworn and unwashed

Original Packaging

Return items in original packaging

Same Condition

Items must be in original condition

Non-Returnable Items

Intimate apparel and swimwear
Items marked as "Final Sale"
Items that have been altered
Gift cards

Return Process

1

Drop off the package at any authorized shipping location
